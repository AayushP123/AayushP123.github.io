# Aayush Pandey Portfolio

Personal software engineering portfolio for [Aayush Pandey](https://aayushp123.github.io/).

The site highlights backend engineering, cloud infrastructure, machine learning research,
and open-source work through a responsive, scroll-driven interface.

## Stack

- Next.js and React
- TypeScript
- vinext and Vite
- GitHub Actions and GitHub Pages

## Local development

```bash
npm install
npm run dev
```

The development server runs at `http://localhost:3000`.

## Build and verify

```bash
npm run build
npm test
```

## Deployment

Every push to `main` builds the application, exports a static version, and publishes it to
GitHub Pages through `.github/workflows/pages.yml`.
